# CampusNest

Student accommodation web application.

Live demo: https://campusnest-sa.netlify.app/index.html

## Tech stack
- HTML
- CSS
- JavaScript
- Firebase (authentication / database / hosting)
- Deployed to Netlify

## Features
- Search and browse student accommodations
- Contact landlords
- Basic authentication via Firebase

## Run locally
1. Clone the repo:
   git clone https://github.com/skumalo0115-commits/CampusNest.git
2. Change into the project directory:
   cd CampusNest
3. If the project uses Node (see package.json), install dependencies:
   npm install
4. For a simple static preview, open `index.html` in your browser or run a local server:
   npx http-server .
   # or
   python -m http.server 3000

## Environment / Secrets
This project likely requires Firebase configuration. Create a `.env` file (or use your Firebase config file) and add the necessary keys. Do NOT commit secrets to the repository.
